## Installation
To run this app locally, follow these steps:

1. Clone the repo locally

```bash
git clone https://github.com/danielkiing3/path_animation.git
cd headspace_onboarding
```
2. Install dependencies:
```bash
flutter pub get
```
3. Run the app:
```bash
flutter run
```
